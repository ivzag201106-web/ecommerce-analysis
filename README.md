# 🛒 Анализ E-commerce платформы Olist

## Описание проекта

Комплексный анализ транзакционных данных бразильского маркетплейса Olist. Проект охватывает полный аналитический цикл: от обработки сырых данных средствами SQL до визуализации ключевых показателей в интерактивном дашборде Power BI.

## Стек инструментов

- **SQL** (Beekeeper Studio / SQLite) — очистка данных, агрегация, аналитические запросы
- **Python** (pandas, matplotlib) — валидация данных, построение графиков
- **Power BI** — интерактивный дашборд, DAX-меры

## Датасет

Olist Brazilian E-commerce Dataset — 5 связанных таблиц, ~89 000 строк транзакций.

| Таблица | Описание |
|---|---|
| Orders | Заказы и статусы доставки |
| Payments | Платёжные транзакции |
| OrderItems | Позиции заказов |
| Products | Товарный каталог |
| Customers | Клиентская база |

## Ключевые показатели

| Метрика | Значение |
|---|---|
| Суммарный оборот | 24 млн бразильских реалов |
| Количество заказов | 50 000 |
| Средний чек | 272.69 реала |
| Доля просроченных заказов | 7.67% |

## Основные выводы

**Товарный портфель** — топ-3 категории по объёму выручки: игрушки, товары для сада, товары для здоровья и красоты. Категория игрушек значительно опережает остальные по суммарному обороту.

**География** — наибольшая концентрация клиентской базы сосредоточена в штате SP (Сан-Паулу) — 38 000 клиентов, что отражает высокую экономическую активность региона. Наибольшее среднее время доставки зафиксировано в северных штатах страны.

**Операционная эффективность** — 92.33% заказов доставлены в срок. Доминирующий платёжный инструмент — кредитная карта (66 000 транзакций), однако наибольший средний чек показывают транзакции через ваучеры (310 реалов).
